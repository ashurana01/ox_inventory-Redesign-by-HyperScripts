# Hyper UI - ox_inventory

Modern free redesign for ox_inventory.

> Clean visuals. Better UX. Same functionality.
> HUD in inventory is static to make it work use your framework export.

## Preview

A cleaner and more modern redesign of ox_inventory built to improve readability, layout, and overall user experience without changing core functionality.

## Features

- Modern inventory redesign
- Cleaner item slots
- Better spacing
- Improved readability
- Lightweight
- Free to use

## Installation

1. Download
2. Install ox_inventory
3. ensure ox_inventory
4. Restart server
